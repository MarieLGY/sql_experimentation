On Tuesday, October 3rd 2017, we organized an SQL experimentation. 70 computer science students parcipated, and answered 10 SQL related challenges. They were divided into two groups:

  • The first group had access to a classic SQL editor with basic functionnalities
  • The second group had the same editor but with an additional option: SQL query completion
